# Qanun Design System Skill

## Colours
- Navy (primary bg): #0B1829
- Gold (Research accent): #C4922A / Qanun Gold: #B8974A
- Teal (Studio accent): #0F7A5F / Studio Teal: #0F6E56
- NO gradients ever

## Typography
- Font: Inter
- Weights: 400 (body), 600 (headings) — no other weights
- Brand typography pairing: Fraunces/Cormorant Garamond (marketing), Instrument Sans (UI)

## Icons
- Library: Lucide React
- strokeWidth: 1.5 (always)
- Size: contextual, but consistent within component groups

## Constraints
- Max border-radius: 16px
- Aesthetic: dark, minimal, editorial
- Benchmarks: Legora, Harvey
- No cheerful/rounded SaaS aesthetics
- No generic Tailwind defaults — every component must feel intentional

## Component Patterns
- Cards: dark bg, subtle border, generous padding
- Buttons: filled (gold/teal for primary), ghost (for secondary)
- Tables: no zebra striping, clean grid lines, compact
- Modals: dark overlay, centered, max-width 640px
